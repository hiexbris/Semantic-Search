Blockchain represents a decentralized and secure digital ledger that records transactions across a network of computers. It ensures transparency, immutability, and tamper resistance, making data manipulation difficult. The underlying technology for cryptocurrencies like bitcoin, blockchain has applications beyond finance in areas such as supply chain management and smart contracts.

Data stored on the blockchain is generally considered incorruptible due to its distributed nature, with each node having a copy of the blockchain and data quality being maintained by massive database replication and computational trust. Each block typically contains a hash pointer linking it to the previous block, a timestamp, and transaction data, making changes to the data in any given block difficult without altering all subsequent blocks, which requires collusion from the network majority.

Blockchains are secure by design and have high Byzantine fault tolerance, allowing for decentralized consensus to be achieved. A node is a computer that connects to a cryptocurrency network and supports it through relaying transactions, validation, or hosting a copy of the blockchain. The network requires minimal structure to share transactions, with an ad hoc decentralized network of volunteers sufficient for broadcasting messages on a best-effort basis.
